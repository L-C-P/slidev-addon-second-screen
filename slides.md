# Second Screen Addon Demo

Test presentation for the second screen addon

---

## Features

- Open presentation on second screen
- Fullscreen prompt with keyboard shortcuts
- Visual status indicator
- Auto-detect manual close

---

## How to Test

1. Open presenter mode (press `P` or click presenter button)
2. Look for the new button in the navigation bar
3. Click the screen icon to open on second screen
4. The second screen will open on the monitor where the presenter is NOT running
